# Digitally Controlled Oscillator (DCO)

A Digitally Controlled Oscillator (DCO) designed with a frequency range of 3.33 MHz to 5.53 MHz and an estimated power consumption of approximately 25 µW. 

This project was developed as my final year undergraduate project in Electrical Engineering. A key highlight of this design is that it was **entirely developed using open-source EDA tools**, demonstrating the viability of open-source flows for custom integrated circuit design.

**Full Manuscript:** [Read the thesis here](https://lume.ufrgs.br/handle/10183/295344)

## Specifications
* **Frequency Range:** 3.33 MHz – 5.53 MHz
* **Power Consumption:** ~25 µW
* **Tools Used:** NgSpice, Xschem, Magic and Netgen
* **Technology Node:** SKY130
